## Βήμα 2: Φορτώστε το αρχείο αιτήματος OFX
Σε αυτό το βήμα, θα φορτώσουμε το αρχείο αιτήματος OFX που θέλετε να μετατρέψετε. Θα υποθέσουμε ότι έχετε το αρχείο αποθηκευμένο σε έναν κατάλογο στον υπολογιστή σας.
```csharp
// Καθορίστε τον κατάλογο προέλευσης όπου βρίσκεται το αρχείο αιτήματός σας OFX
string sourceDir = "Your Source Directory";
// Φορτώστε το έγγραφο αίτησης OFX από το καθορισμένο αρχείο
OfxRequestDocument document = new OfxRequestDocument(sourceDir + @"bankTransactionReq.sgml");
```
## Εξήγηση
- sourceDir: Αυτή είναι η διαδρομή καταλόγου όπου βρίσκεται το αρχείο αιτήματός σας OFX. Αντικαθιστώ`"Your Source Directory"` με την πραγματική διαδρομή.
- OfxRequestDocument: Αυτή η κλάση χρησιμοποιείται για τη φόρτωση του αρχείου αιτήματος OFX σε ένα αντικείμενο εγγράφου για περαιτέρω επεξεργασία.
## Βήμα 3: Μετατρέψτε το αρχείο αιτήματος OFX σε Αίτημα OFX V2
Μόλις φορτωθεί το έγγραφο, μπορείτε να το μετατρέψετε σε OFX Request V2. Αυτό περιλαμβάνει την αποθήκευση του εγγράφου στη νέα μορφή.
```csharp
// Καθορίστε τον κατάλογο εξόδου όπου θα αποθηκευτεί το αρχείο που έχει μετατραπεί
string outputDir = "Your Output Directory";
// Αποθηκεύστε το έγγραφο σε μορφή OFX Request V2
document.Save(outputDir + @"bankTransactionReq.xml", OfxVersionEnum.V2x);
```
## Εξήγηση
-  outputDir: Αυτή είναι η διαδρομή καταλόγου όπου θα αποθηκευτεί το αρχείο OFX που έχει μετατραπεί. Αντικαθιστώ`"Your Output Directory"` με την πραγματική διαδρομή.
-  Μέθοδος αποθήκευσης: Το`Save` μέθοδος χρησιμοποιείται για την αποθήκευση του εγγράφου στην καθορισμένη μορφή. Η δεύτερη παράμετρος καθορίζει την έκδοση OFX στην οποία θέλετε να μετατρέψετε το αρχείο (OFX V2 σε αυτήν την περίπτωση).
